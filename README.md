
A Flutter quiz application testing general knowledge with true/false questions.

**Project Focus:**
State management with Stateful/Stateless Widgets
Object-Oriented Programming concepts in Dart
Code organization using Classes and Objects

**Technical Concepts:**
Widget refactoring and extraction
Dart class constructors and modifiers
List handling and data management

**Question Data Structure:**
Pre-defined true/false questions covering various topics
Sample questions include animal facts, human anatomy, historical trivia
